# NHS Prescribing Analysis

## Project Overview

This project analyses NHS-style prescribing data to identify cost trends, prescribing patterns, regional differences, and opportunities for improved reporting and decision-making.

## Objectives

- Analyse prescribing costs and volume
- Identify high-cost medicines or treatment categories
- Compare prescribing patterns across regions
- Create insights useful for healthcare service planning
- Demonstrate SQL, Excel, Power BI, and analytical reporting skills

## Tools Used

- SQL
- Microsoft Excel
- Power BI
- GitHub

## Key Questions

1. Which medicines or categories have the highest prescribing cost?
2. How do prescribing trends change over time?
3. Which regions show the highest prescribing activity?
4. Are there cost-saving opportunities?
5. What insights can support NHS-style operational reporting?
