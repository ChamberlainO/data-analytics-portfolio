# Mental Health Prescribing Analysis

## Project Overview

This project analyses mental health prescribing data to identify prescribing trends, medication usage patterns, cost implications, and regional variations in mental health treatment.

The project was inspired by the NHS Step Up virtual career challenge: Turning Data into Insight – Building Awareness of Mental Health Prescribing.

## Objectives

- Analyse prescribing trends for mental health medications
- Explore variations in prescribing patterns
- Identify high-cost medication categories
- Generate insights relevant to healthcare service planning
- Demonstrate healthcare analytics and data science skills using Python

## Skills Demonstrated

- Data Cleaning
- Exploratory Data Analysis
- Healthcare Analytics
- Statistical Analysis
- Data Visualisation
- Insight Generation
- Python Programming

- Python

## Project Structure

- `data/` → raw datasets
- `notebooks/` → Python notebooks
- `visuals/` → charts and visual outputs
- `reports/` → project summaries and findings

## Key Questions

1. Which mental health medications are prescribed most frequently?
2. Are there observable prescribing trends over time?
3. Which regions or practices show higher prescribing rates?
4. What are the cost implications of mental health prescribing?
5. What insights can support awareness and healthcare planning?
